云计算与个人博客搭建

本文是我在学习《云计算原理与应用》课程中的实践总结，主要内容包括使用Hexo搭建静态博客、配置GitHub Pages自动化部署、以及使用Butterfly主题进行定制化配置。

一、为什么选择Hexo？

Hexo是一个快速、简洁且高效的博客框架，支持Markdown语法，可以轻松生成静态页面，非常适合搭配GitHub Pages使用。

二、部署流程总结

  1. 环境准备：安装Node.js、Git、Hexo CLI
  2. 博客初始化：hexo init <blog-name>
  3. 主题安装：npm install hexo-theme-butterfly
  4. 配置部署：修改_config.yml，安装hexo-deployer-git
  5. 发布文章：hexo new "文章标题"，编辑后执行hexo g -d

三、云计算与静态网站托管

通过将静态博客部署到GitHub Pages或阿里云OSS等服务，我们实际上是在利用云计算中的PaaS（平台即服务） 或静态网站托管能力，实现低成本、高可用的网站部署。
